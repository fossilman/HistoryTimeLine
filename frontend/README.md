# History Timezone Frontend

前端 React 应用

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

## 构建

```bash
npm run build
```

## 预览构建结果

```bash
npm run preview
```

## 环境变量

创建 `.env` 文件（可选）：

```bash
VITE_API_BASE_URL=http://localhost:3000/api/v1
```
